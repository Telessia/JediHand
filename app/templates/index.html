JEDi HAND

Description

Application JediHand réalisée par Quentin Colras et Xavier Fostier, projet Semestre 10 Master 2 TNSID UPHF Valenciennes

Cette application permet d'exécuter des actions sur la machine hôte en effectuant des gestes de la main comme par exemple : monter/diminuer le volume système, lancer un lien web ou une application/script, arrêter/relancer une vidéo Youtube etc...

Le menu sur la gauche permet d'accèder aux différents modules de l'application : le rouage permet d'affecter une commande à un signe de la main, la caméra permet d'avoir un retour vidéo lors de la detection des signes et permet de prendre un nouveau signe en photo, l'icone d'image permet l'affichage de tout les signes et leurs commandes associées, enfin vous pouvez également vous rendre sur le Github du projet pour y inspecter/télécharger le code source.
